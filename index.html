Instructions

Currently it is possible to control the planes around. To order a plane to do something, type its identifier followed by the plane's commands. Commands are a combination of one verb and an argument, followed by a separator (comma). If the command is invalid the plane will not perform the command. The verbs and their respective argument(s) are: 

  • a
    altitude, expects one number as parameter, the target altitude in feet.
  • s
    speed, expects one number, the target speed in knots.
  • c
    cleared to, expects one string, the name of the navaid to which the aircraft is cleared (direct, except if via is specified.)
  • v
    via, expects a string of waypoints in the order they should be passed before the cleared waypoint, separated by spaces.
  • h
    fly heading, expects one number, the heading the aircraft shall fly.
Each command should be followed by a ',' (comma).

An example command is (as entered by the user): 

ASD123,a3000,s200,cARS,vKOGAV ELTOK<ENTER>
Read out, this command is: 
ASD123, altitude 3000, speed 200, cleared to ARS, via KOGAV ELTOK

If you make a mistake while writing the commands you can use <BACKSPACE> as usual. <ESC> clears the whole command.
